Caso practico 5

Descripcion de los archivos:
    - main.py: pruebas con los dos conjuntos de datos
    - Presentacion.pdf: presentacion del caso
    - README.txt: este archivo

Descripcion de las carpetas:
    - conjuntos: implementacion del caso
        - conjuntos.py: archivo con la implementacion para sacar los conjuntos frecuentes
        - reglas.py: archivo con la implmentacion para sacar las reglas
    - data: carpeta donde se guardan los archivos con las transacciones
        - retail.dat: archivo con las transacciones de un supermercado
        - test.dat: archivo con transacciones de pruebas
    - persistencia: carpeta con la implementacion para leer los archivos con las transacciones
        - persistencia.py: archivo con la implementacion para leer las transacciones
